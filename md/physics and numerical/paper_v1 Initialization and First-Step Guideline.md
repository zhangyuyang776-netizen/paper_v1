# paper_v1 Initialization and First-Step Guideline (Final)

## 1. 文件定位

本文档用于规定 `paper_v1` 项目中：

1. 初始几何与初始场的构造方式；
2. 文献 Eq. (2.20)、Eq. (2.21)、Eq. (2.22) 在新代码中的落实方式；
3. 首时间步（first step）的 predictor / outer / inner 初值设置；
4. 初始化中哪些部分直接遵循文献，哪些部分是 `paper_v1` 的工程化扩展。

本文档是 `paper_v1` 初始化专题的正式约束文件。  
后续讨论不得再回到旧代码修补思路。

## 2. 总原则

### 2.1 文献优先原则

对于初始化问题，凡文献已有明确说明者，原则上按文献执行。  
本专题中，文献已明确给出：

- 气相初始温度场：Eq. (2.20)
- 气相初始组分场：Eq. (2.21)
- 液相初始温度与组成：Eq. (2.22)

因此，`paper_v1` 的严格基线初始化应以这些公式为基础构造。

### 2.2 时间原点约束

`paper_v1` 的求解时间从

\[
t = 0
\]

开始。

这里需要明确区分两个概念：

- **solver time**：真正进入时间推进器的时间变量，从 `0` 开始；
- **initialization smoothing parameter**：文献中的 `t_0`，仅用于构造初始化温度/组分平滑剖面，不作为求解器时间原点，也不作为一个先行推进的时间层。

因此，本项目不采用 “从 \(t=t_0\) 开始积分” 的处理，而采用：

- **从 \(t=0\) 开始求解**
- **用 \(t_0\) 代入 Eq. (2.20)、Eq. (2.21) 生成初始场**

### 2.3 与 outer / inner 主线的一致性

初始化后的第一个时间步，仍然必须遵守已定的两层结构：

- `outer`：负责 `a/Rd`、`dot_a`、网格、`v_c`、remap、时间步接受/拒绝
- `inner`：负责 fixed-geometry 下的 liquid bulk + interface block + gas bulk 的耦合非线性求解

不得为首步单独发明另一套主算法。

## 3. 初始时间层与初始几何

### 3.1 初始时间层

正式规定：

\[
t^0 = 0
\]

### 3.2 初始液滴半径

正式规定：

\[
a^0 = a_0
\]

其中 `a0` 由配置文件 `geometry.a0` 显式给出。

### 3.3 初始网格

初始几何 `G^0` 按三段网格正式规则，由 `a^0 = a0` 直接构造：

1. 液相段：\([0, a_0]\)，均匀网格
2. 近界面气相段：\([a_0, 5a_0]\)，均匀网格
3. 远场气相段：\([5a_0, r_{end}]\)，固定几何扩张网格

第三段仍采用已定工程化主线，不随首步特殊改写。

## 4. 文献式初始化参数 `t0`

### 4.1 定义

`paper_v1` 中定义一个初始化平滑参数：

\[
t_0 > 0
\]

该参数仅用于代入文献 Eq. (2.20)、Eq. (2.21)，生成平滑的初始气相温度场和初始气相组分场。

### 4.2 物理/数值含义

`t0` 的作用是避免直接使用尖锐阶跃初值，从而减小初始数值刚性。  
它不是一个真实推进过的时间层，也不代表系统已经演化到 `t=t0`。

### 4.3 取值约束

应满足文献所强调的“小时间平滑”思想，即：

\[
t_0 \ll \frac{a_0^2}{D_{T,g,\infty}}
\]

其中

\[
D_{T,g,\infty} = \frac{k_{g,\infty}}{c_{p,g,\infty}\rho_{g,\infty}}
\]

为远场气相热扩散率。

首版建议取文献量级：

\[
t_0 \sim 10^{-5}\ \text{s}
\]

但最终值应由配置文件显式给出，不允许隐式默认。

## 5. 液相初始场（直接按文献 Eq. (2.22)）

### 5.1 液相初始温度

在液滴内部 \(0 \le r \le a_0\)，初始温度取均匀场：

\[
T_\ell(r,0) = T_{d0}
\]

其中 `Td0` 由配置文件显式给出。

### 5.2 液相初始组成

#### 单组分液滴首版

\[
Y_{\ell,\text{full}}(r,0) = [1.0]
\]

#### 多组分液滴升级版

\[
Y_{\ell,i}(r,0) = Y_{\ell,i,0}
\]

即液滴内部初始组成为均匀场。

### 5.3 实现约束

液相状态层中应保存 full-order 组成；  
reduced species 仅是 unknown layout 层面的裁剪，不改变物理状态存储。

---

## 6. 气相初始温度场（按文献 Eq. (2.20)）

### 6.1 公式

在 \(r \ge a_0\) 的气相区域，初始温度场定义为：

\[
T_g(r,0)
=
T_\infty
+
\frac{a_0}{r}(T_{d0}-T_\infty)
\,
\mathrm{erfc}
\left(
\frac{r-a_0}{2\sqrt{D_{T,g,\infty}t_0}}
\right)
\]

其中

\[
D_{T,g,\infty}
=
\frac{k_{g,\infty}}{c_{p,g,\infty}\rho_{g,\infty}}
\]

### 6.2 远场热扩散率的计算

远场物性必须使用远场气相状态：

- \(T_\infty\)
- \(Y_{g,\infty}\)
- \(P_\infty\)

通过 Cantera 计算得到：

- \(k_{g,\infty}\)
- \(c_{p,g,\infty}\)
- \(\rho_{g,\infty}\)

再构造 \(D_{T,g,\infty}\)。

### 6.3 解释

该剖面是文献明确给出的平滑初始化方式。  
因此，严格文献基线初始化下，不允许将气相初温场简化为全域常数 \(T_\infty\)。

---

## 7. 气相初始组分场（按文献 Eq. (2.21)）

### 7.1 一般公式

在 \(r \ge a_0\) 的气相区域，初始组分场定义为：

\[
Y_{g,i}(r,0)
=
Y_{g,i,\infty}
+
\frac{a_0}{r}(Y_{g,i,0}-Y_{g,i,\infty})
\,
\mathrm{erfc}
\left(
\frac{r-a_0}{2\sqrt{D_{T,g,\infty}t_0}}
\right)
\]

其中：

- \(Y_{g,i,\infty}\)：远场气相质量分数
- \(Y_{g,i,0}\)：界面处初始气相质量分数的给定值

### 7.2 单组分液滴首版

对于蒸发物种，其界面初始气相质量分数按文献思路取一个很小正数：

\[
Y_{g,\text{vap},0} = \varepsilon_{\text{vap}}
\]

推荐量级：

\[
\varepsilon_{\text{vap}} \sim 10^{-8}
\]

其余非凝组分按质量分数和为 1 的条件补齐。

### 7.3 多组分液滴升级版（工程化扩展）

文献对多组分情形下的 `Y_{g,i,0}` 并未逐项给出，因此 `paper_v1` 作如下工程化扩展：

对于所有可凝组分 \(c\)：

\[
Y_{g,c,0} = \varepsilon_{\text{vap},c}
\]

其中 \(\varepsilon_{\text{vap},c}\) 为小正数，默认量级 \(10^{-8}\)。

对于所有非凝组分 \(j\)，按远场非凝组分比例缩放补齐：

\[
Y_{g,j,0}
=
\alpha\,Y_{g,j,\infty}
\]

其中

\[
\alpha
=
\frac{1-\sum_c Y_{g,c,0}}
{1-\sum_c Y_{g,c,\infty}}
\]

这样可保证：

1. 全组分质量分数和为 1；
2. 非凝组分保持与远场一致的相对比例；
3. 可凝组分在界面处具有极小但非零的初始蒸汽种子；
4. 与文献“平滑启动、降低刚性”的意图一致。

### 7.4 约束说明

这一多组分扩展属于 `paper_v1` 工程化规则，不属于文献逐字给出的公式。  
文档和代码中必须将其标记为 “project extension”，不得伪装成文献原式。

## 8. 界面初始变量块

已定 unknown strategy 规定：

- 单组分首版：
  \[
  \text{interface\_A} = [T_s, Y_{s,g,\mathrm{red}}, mpp]
  \]

- 多组分升级版：
  \[
  \text{interface\_B} = [Y_{s,\ell,\mathrm{red}}, T_s, Y_{s,g,\mathrm{red}}, mpp]
  \]

因此，初始化时也必须显式构造界面块初值。

### 8.1 初始界面温度

取：

\[
T_s^0 = T_{d0}
\]

这是与液滴内部均匀初温一致的定义，也与文献式平滑温度场在 \(r=a_0\) 处的连续性相容。

### 8.2 初始液相界面组成

#### 单组分首版

\[
Y_{s,\ell,\text{full}}^0 = [1.0]
\]

#### 多组分升级版

\[
Y_{s,\ell,i}^0 = Y_{\ell,i,0}
\]

### 8.3 初始气相界面组成

取式 (2.21) 在 \(r=a_0\) 处的值，即：

\[
Y_{s,g,i}^0 = Y_{g,i,0}
\]

注意：

- 这里应取文献式初始化所定义的“小蒸汽种子”界面值；
- 不应在初始化阶段强行把 \(Y_{s,g}^0\) 设为界面相平衡值。

这是一个关键约束：  
**初始化界面气相组分来自 Eq. (2.21) 的给定值，而不是来自界面平衡方程的代数解。**

### 8.4 初始蒸发质量通量

正式规定：

\[
mpp^0 = 0
\]

文献在初始化公式中未明确给出 `mpp^0` 的非零表达，因此首版采用零值作为首步 inner 的一致初猜。

## 9. 初始界面速度与首步 predictor

### 9.1 初始界面速度

正式规定：

\[
\dot a^0 = 0
\]

理由如下：

1. 文献初始化部分并未明确给出一个非零初始界面退缩速度公式；
2. `paper_v1` 已确定 `mpp` 为独立界面 unknown，其物理值应由首步 inner 解产生，而不是在初始化时预设为一个未经验证的非零量；
3. 采用 \(\dot a^0 = 0\) 可使首步 predictor 保持最简单一致的形式。

### 9.2 首步 predictor

首时间步第 0 轮 predictor 仍采用统一公式：

\[
\dot a^{(0)} = \dot a^0
\]

\[
a^{(0)} = a^0 + \Delta t\,\dot a^0
\]

由于 \(\dot a^0 = 0\)，故首步自动退化为：

\[
a^{(0)} = a_0
\]

### 9.3 解释

因此，首步 predictor 不需要单独改写。  
真正需要文献约束的是 **初始温度场和初始组分场的构造**，而不是 predictor 公式本身。

## 10. 首步 outer / inner 初值设置

### 10.1 首步 outer 第 0 轮

初始输入为：

- \(t^0 = 0\)
- \(a^0 = a_0\)
- \(\dot a^0 = 0\)
- 初始网格 \(G^0\)
- 初始状态 \(U^0\)

据此构造首步第 0 轮 outer 几何状态：

- \(a^{(0)} = a_0\)
- \(\dot a^{(0)} = 0\)
- \(G^{(0)} = G^0\)

### 10.2 首步控制面速度

按已定三段动网格规则，当 \(\dot a^{(0)} = 0\) 时：

\[
v_c^{(0)}(r)=0
\]

即首轮 outer 几何上没有网格运动。

### 10.3 首步 remap

首步第 0 轮 remap 仍调用正式 remap 接口，但由于几何未变，结果应退化为恒等映射：

\[
\text{old\_state\_on\_current\_geometry}^{(0)} = U^0
\]

不建议为首步单独绕过 remap 模块，避免主流程分裂。

### 10.4 首步 inner 初值

正式规定：

\[
U_{\mathrm{init}}^{(0)} = U^0
\]

即将初始化构造出的状态直接作为首轮 inner 的初值。

若首步 outer 后续还发生第 \(k>0\) 轮迭代，则仍按正式主线：

\[
U_{\mathrm{init}}^{(k)} = U^{(k-1)}
\]

## 11. 允许与不允许的首步特殊处理

### 11.1 允许的数值保护

首步允许采用更保守的 solver policy，例如：

- 更小的 `dt_start`
- 更保守的 outer 欠松弛
- 更积极的 inner line search
- 首步失败后优先缩小时间步

这些属于数值求解策略，不改变物理与离散结构。

### 11.2 不允许的处理

首步不得采用以下方式“偷偷改模型”：

1. 不得先做一个伪稳态界面求解，再把结果伪装成正式初始层；
2. 不得用猜测的非零 `mpp_init` 推进初始几何；
3. 不得把 `Rd/a` 临时塞回 inner 主未知量；
4. 不得为首步定义不同于正式 residual structure 的方程组；
5. 不得把界面气相初值直接强制设为平衡值来替代 Eq. (2.21) 初始化。

## 12. 初始化专题最终约束汇总

### 12.1 直接来自文献的部分

以下内容视为文献明确给出的初始化主线：

1. 液滴内部初始温度均匀：
   \[
   T_\ell(r,0)=T_{d0}
   \]

2. 液滴内部初始组成均匀：
   \[
   Y_{\ell,i}(r,0)=Y_{\ell,i,0}
   \]

3. 气相初始温度场按 Eq. (2.20)：
   \[
   T_g(r,0)
   =
   T_\infty
   +
   \frac{a_0}{r}(T_{d0}-T_\infty)
   \,
   \mathrm{erfc}
   \left(
   \frac{r-a_0}{2\sqrt{D_{T,g,\infty}t_0}}
   \right)
   \]

4. 气相初始组分场按 Eq. (2.21)：
   \[
   Y_{g,i}(r,0)
   =
   Y_{g,i,\infty}
   +
   \frac{a_0}{r}(Y_{g,i,0}-Y_{g,i,\infty})
   \,
   \mathrm{erfc}
   \left(
   \frac{r-a_0}{2\sqrt{D_{T,g,\infty}t_0}}
   \right)
   \]

5. \(t_0\) 仅为小时间平滑参数，服务于初始化剖面构造。

### 12.2 `paper_v1` 工程化补充的部分

以下部分属于项目正式工程化约束：

1. 求解时间从 \(t=0\) 开始，而不是从 \(t=t_0\) 开始；
2. 初始界面变量块必须显式构造；
3. 初始界面气相组分取 \(Y_{g,i,0}\)，而不是界面相平衡值；
4. 初始蒸发质量通量：
   \[
   mpp^0 = 0
   \]
5. 初始界面速度：
   \[
   \dot a^0 = 0
   \]
6. 首步 predictor 沿用正式统一公式，不另开特殊分支；
7. 多组分情形下的 \(Y_{g,i,0}\) 采用 “可凝组分小种子 + 非凝组分比例补齐” 的扩展规则。
